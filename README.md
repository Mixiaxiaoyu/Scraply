# 拾页 Scraply

`拾页` 是一个把多张照片自动整理成纸质手账拼贴图的 Skill。

它适合把旅行、散步、城市、自然、人物、小动物、风景等照片，做成一张已经排好版的双页手账图。生成结果不是简单拼图，而是带有纸张、贴纸、票据、胶带、手写标题和整体主题感的成品画面。

## 名称信息

- 中文名：`拾页`
- 英文名：`Scraply`
- Skill 标识名：`scraply`
- 实际 Skill 名：`scraply`

## 这个 Skill 能做什么

- 把上传的多张照片做成一张完整的双页手账图
- 自动根据照片内容匹配贴纸、票据、标签、手写文案和装饰
- 自动控制页面结构，不只是把图片并排摆上去
- 提供两个版本：
  默认经典版：更像稳定、干净、成熟的参考成品
  `@S` 丰富随机版：本体、边缘配件、纸张元素和装饰会更活跃
- 自动随机标题位置，不再总是固定在左边

## 核心特点

- 默认生成“完美经典版”：蓝色切割垫、干净双页本、白边照片、蓝色手写标题、票据、贴纸、少量夹子和笔
- 输入 `@S` 后开启丰富随机版：透明封套、活页环、吊坠、回形针、标签页、书签带、夹子等会更加多变
- 标题位置随机：会在左上、右上、偏中上、跨中缝、主图上方、文字页等位置里选，不再总在左侧
- 文案随机：不会总是重复同一套标题、气泡和清单
- 主题自动匹配：城市会偏路线/票据/位置图标，自然会偏植物/动物/水波/草地等
- 会尽量把你上传的照片都用进去

## 适用场景

- 旅行照片整理
- 日常散步、露营、徒步、看展、小型出游
- 城市街景、建筑、水岸、古镇、自然风景
- 动物、花草、牧场、湖边、山野主题
- 人像、情侣照、纪念照、情绪感照片
- 想把一组图快速做成可分享的“手账成品图”

## 使用建议

- 最推荐：`3-5 张图`
- `1-2 张图` 也能做，但画面信息量会更轻
- `5-8 张图` 适合更丰富的双页排版
- 如果照片太多，页面会更密，经典干净感会相对下降

如果你想要效果最稳、最好看，优先建议准备 `3-5 张图`，题材尽量围绕一个统一主题，比如同一段旅行、同一次散步、同一种情绪或同一组自然场景。

## 怎么使用

### 默认经典版

直接上传照片并调用 Skill 即可，不需要额外加命令。

示例：

```text
Use $scraply to turn these uploaded images into one finished scrapbook spread.
```

### 丰富随机版

如果想让本体更活、边缘装饰更多、手账本身更有变化，就在请求里加上 `@S`。

示例：

```text
Use $scraply @S to turn these uploaded images into one finished scrapbook spread.
```

说明：
`@S` 只是模式开关，不会被画进图里。

## 当前版本的优化点

- 默认版和丰富版分离，不再混在一起
- 标题位置随机，不再总放左边
- 文案更灵活，不再总是重复固定句子
- 丰富版只在 `@S` 时启用，不会污染默认经典版

## 输出风格总结

- 默认版：稳、干净、像参考图、适合长期批量使用
- `@S` 版：更像收藏册，更有手作感和本体变化，适合想要一点惊喜的时候
